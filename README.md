# MARS ROVER Exercies
## How to run
    npm install
    npm run build
    npm start
The webserver will be available at **http://0.0.0.0:8000**

### or using Docker
    npm run build:docker
    docker run -p 3000:3000 -p 8000:8000 mars-rover
    
## Testing
    npm run test
    
## Generating documentation
    npm run build:docs
